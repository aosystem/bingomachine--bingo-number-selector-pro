# Bingo Number Selector Pro 🎲

A professional-grade random number generator and bingo caller built with **Flutter**. This app is streamlined for maximum speed and reliability, making it the perfect tool for events, classrooms, and any situation requiring fair random selection.

## 🌟 Key Features
- **Pure Selection Logic:** Focused on fast, reliable number drawing without unnecessary distractions.
- **Large Display:** Clear and easy-to-read numbers, ideal for hosting events.
- **Full History:** Track every number called in the current session.
- **Global Support:** Localized in **50+ languages** for users across the globe.

## 💡 Best For...
- **Professional Bingo Hosting:** Reliable and simple to operate.
- **Classroom Activities:** Selecting student numbers or randomizing tasks.
- **Event Raffles:** Fairly choosing winners from a set range of numbers.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** High-speed randomization and state preservation for session history.

*Note: This repository contains the core logic and source code. Assets (images, sound effects, and localization files) are excluded to protect intellectual property.*
